Nitrux 6.0.0 combines the latest software updates, bug fixes, and performance improvements with ready-to-use hardware support. This release continues the evolution of our immutable foundation.

### Key Enhancements
* **Kernel:** Includes Linux Kernel 6.18 LTS for enhanced hardware compatibility and stability.
* **NX AppHub:** Updated foundations for rootless, declarative app management.
* **MauiKit:** Latest improvements to the modular UI framework for a more refined experience.
* **Integrity:** Enhanced system-wide security policies and filesystem optimizations.
